# Categories (Typologies)

Les **Typologies** són categories per organitzar els teus projectes.

## Crear una categoria

1. Al panell, ves a **Content Manager** > **Typologies**
2. Clica **Create new entry**
3. Omple els camps:

| Camp | Descripció |
|------|------------|
| **Name** | Nom de la categoria (obligatori i únic) |
| **Slug** | S'autogenera (URL) |
| **Subtitle** | Descripció de la categoria |
| **Image** | Imatge o icona representativa |

## Exemples de categories

Pots crear categories com:

- Disseny Web
- UX/UI
- Branding
- Fotografia
- Desenvolupament

## Vincular projectes

Un cop creada la categoria, pots vincular projectes de dues maneres:

### Des de la categoria:
1. Edita la categoria
2. A la secció **Projects**, selecciona els projectes

### Des del projecte:
1. Edita el projecte
2. A la secció **Typologies**, selecciona les categories

:::tip Consell
És més pràctic assignar categories des de cada projecte individualment.
:::

## Desar i publicar

1. Clica **Save**
2. Clica **Publish**
