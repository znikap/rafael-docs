# Pàgines

Les **Pàgines** són contingut flexible que pots crear amb diferents blocs.

## Crear una nova pàgina

1. Al panell, ves a **Content Manager** > **Pages**
2. Clica **Create new entry**
3. Omple els camps:

| Camp | Descripció |
|------|------------|
| **Title** | Nom de la pàgina (obligatori i únic) |
| **Slug** | S'autogenera a partir del títol (URL de la pàgina) |
| **Subtitle** | Text descriptiu curt (opcional) |

## Afegir contingut amb blocs

La secció **Content** és una zona dinàmica on pots afegir blocs:

1. Clica el botó **+ Add a component**
2. Tria el bloc que vulguis:
   - **Text** - Text simple
   - **Text amb CTA** - Text amb botó d'acció
   - **Text + Foto** - Text al costat d'una imatge
   - **Images** - Galeria d'1 a 3 imatges
   - **Contact Form** - Formulari de contacte

3. Omple els camps del bloc
4. Pots **arrossegar** els blocs per reordenar-los
5. Pots **eliminar** blocs amb la icona de paperera

## Exemple de pàgina

Una pàgina típica podria tenir:

```
1. Bloc Text amb CTA (introducció amb botó)
2. Bloc Images (galeria de 3 fotos)
3. Bloc Text + Foto (explicació amb imatge)
4. Bloc Contact Form (formulari al final)
```

## Publicar la pàgina

1. Quan estigui llesta, clica **Save**
2. Després clica **Publish**
3. Els canvis es veuran al web automàticament

:::tip Important
Recorda: després de publicar, pot trigar uns segons a actualitzar-se al lloc web públic.
:::
