# Pàgina de Serveis

La **Services Page** és la pàgina que mostra tots els teus serveis.

## Accedir

1. Al panell, ves a **Content Manager**
2. A la secció **Single Types**, clica **Services Page**

## Camps disponibles

### Capçalera (Header)

| Camp | Descripció |
|------|------------|
| **Title** | Títol de la pàgina |
| **Subtitle** | Subtítol descriptiu |

### Contingut

Blocs disponibles per aquesta pàgina:

| Bloc | Descripció |
|------|------------|
| **Text** | Text explicatiu |
| **Text amb CTA** | Text amb botó d'acció |
| **Text + Foto** | Text al costat d'una imatge |
| **Images** | Galeria d'imatges |
| **Services Archive** | Llista de serveis |

## Bloc Services Archive

Aquest bloc mostra una llista de serveis:

1. Afegeix el bloc **Services Archive**
2. Afegeix cada servei:
   - **Title** - Nom del servei
   - **Subtitle** - Descripció
   - **Image** - Icona (opcional)

## Desar canvis

Clica **Save** per desar els canvis.
