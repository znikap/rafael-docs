# Projectes

Els **Projectes** són les peces del teu portafoli.

## Crear un nou projecte

1. Al panell, ves a **Content Manager** > **Projects**
2. Clica **Create new entry**
3. Omple els camps principals:

| Camp | Descripció |
|------|------------|
| **Title** | Nom del projecte (obligatori) |
| **Slug** | S'autogenera (URL del projecte) |
| **Cover** | Imatge de portada del projecte |
| **Description** | Descripció breu del projecte |

## Secció "About" del projecte

Cada projecte té una secció d'informació detallada:

| Camp | Descripció |
|------|------------|
| **Briefing** | Context o resum del projecte |
| **Definition** | Què es va fer exactament |
| **Link** | Enllaç extern al projecte (si en té) |
| **Date** | Data de realització |
| **Client** | Nom del client |
| **Data/Tags** | Etiquetes separades per comes |

## Contingut del projecte

Igual que les pàgines, pots afegir blocs a la secció **Content**:

- **Text** - Explicacions
- **Text amb CTA** - Text amb botó
- **Text + Foto** - Imatge amb text
- **Images** - Galeria d'imatges del projecte
- **Contact Form** - Formulari (opcional)

## Assignar categories

A la secció **Services** pots vincular el projecte a categories:

1. Clica el camp **Services**
2. Selecciona una o més categories existents
3. El projecte apareixerà filtrat per aquestes categories

:::tip Consell
Crea primer les categories a **Services** abans d'assignar-les als projectes.
:::

## Publicar el projecte

1. Clica **Save** per desar
2. Clica **Publish** per fer-lo visible al web
