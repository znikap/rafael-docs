# Benvingut a la Guia

Aquesta documentació t'ajudarà a gestionar el contingut del teu lloc web **Rafael**.

## Què pots fer?

Des del panell d'administració (Strapi) pots:

- **Crear i editar pàgines** amb blocs de contingut flexibles
- **Gestionar projectes** del teu portafoli
- **Configurar serveis** i categories
- **Gestionar el menú de navegació**
- **Actualitzar xarxes socials** i informació de contacte
- **Activar el mode manteniment** quan calgui

## Accés al panell

Per accedir al panell d'administració:

1. Ves a: **[rafaelcreativo.com/adminet](https://rafaelcreativo.com/adminet)**
2. Inicia sessió amb les teves credencials
3. Veuràs el panell principal amb totes les opcions

## Estructura del contingut

El teu lloc web utilitza dos tipus de contingut:

| Tipus | Descripció |
|-------|------------|
| **Col·leccions** | Pots crear múltiples entrades (Pàgines, Projectes, Categories) |
| **Tipus únics** | Una sola entrada per a tot el lloc (Portada, Xarxes Socials, Manteniment) |
