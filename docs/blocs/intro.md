# Blocs de Contingut

Els **blocs** són els elements que pots afegir a les pàgines per construir el contingut.

## Com funcionen

1. Quan edites una pàgina o projecte, trobaràs una secció **Content**
2. Clica **+ Add a component** per veure els blocs disponibles
3. Tria el bloc que necessites
4. Omple els camps
5. Pots afegir tants blocs com vulguis

## Accions amb blocs

| Acció | Com fer-ho |
|-------|------------|
| **Afegir** | Clica "+ Add a component" |
| **Reordenar** | Arrossega el bloc amb la icona ≡ |
| **Eliminar** | Clica la icona de paperera 🗑️ |
| **Duplicar** | Alguns blocs permeten duplicar |

## Blocs disponibles

### Blocs de text
- [Text](./text) - Text simple amb títol
- [Text amb CTA](./text-cta) - Text amb botó d'acció
- [Text + Foto](./text-foto) - Text al costat d'una imatge

### Blocs d'imatge
- [Images](./images) - Galeria d'1 a 3 imatges

### Blocs especials
- [Process](./process) - Targetes de procés (només portada)
- [Services](./services) - Mostra de serveis
- [Contact Form](./contact) - Formulari de contacte
