# Text Photo

El bloc **Text + Foto** mostra text al costat d'una imatge.

## Camps

| Camp | Descripció |
|------|------------|
| **Alignment** | Posició de la foto: Esquerra o Dreta |
| **Title** | Títol de la secció |
| **Text** | Contingut escrit |
| **Photo** | Imatge a mostrar |

## Com funciona l'alineació

| Alignment | Resultat |
|-----------|----------|
| **Left** | Foto a l'esquerra, text a la dreta |
| **Right** | Text a l'esquerra, foto a la dreta |

## Exemple

```
Alignment: Right
Title: "El nostre procés"
Text: "Treballem en estreta col·laboració amb els nostres clients..."
Photo: [imatge pujada]
```

## Consells

:::tip Alternança
Si poses diversos blocs Text + Foto seguits, alterna l'alineació (Left, Right, Left...) per crear un disseny dinàmic.
:::

## Mida d'imatge recomanada

- Amplada mínima: 800px
- Format: JPG o PNG
- Pes màxim: 2MB
