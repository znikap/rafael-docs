# Services

Hi ha dos blocs per mostrar serveis:

## Services Front (Portada)

Mostra serveis destacats a la pàgina principal.

### Camps

| Camp | Descripció |
|------|------------|
| **Pretitle** | Text sobre la secció (ex: "SERVEIS") |
| **Services** | Llista de serveis |
| **Button** | Botó que enllaça a la pàgina de serveis |

### Cada servei

| Camp | Descripció |
|------|------------|
| **Title** | Nom del servei |
| **Subtitle** | Descripció breu |
| **Image** | Icona del servei |

### Exemple

```
Pretitle: "ELS NOSTRES SERVEIS"

Service 1:
  Title: "Disseny Web"
  Subtitle: "Llocs web a mida"
  Image: [icona web]

Service 2:
  Title: "Branding"
  Subtitle: "Identitat visual"
  Image: [icona marca]

Button:
  Text: "Veure tots els serveis"
  Page: [selecciona pàgina de serveis]
```

---

## Services Archive (Pàgina Serveis)

Mostra una llista simple de serveis.

:::info Disponibilitat
Aquest bloc només està disponible a la **Pàgina de Serveis**.
:::

### Camps

| Camp | Descripció |
|------|------------|
| **Services** | Llista de serveis |

### Cada servei

| Camp | Descripció |
|------|------------|
| **Title** | Nom del servei |
| **Subtitle** | Descripció |
| **Image** | Icona (opcional) |
