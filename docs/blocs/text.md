# Bloc Text

El bloc **Text** és el més bàsic per afegir contingut escrit.

## Camps

| Camp | Descripció |
|------|------------|
| **Title** | Títol de la secció |
| **Text** | Contingut del paràgraf |
| **TitleSolo** | Mostrar només el títol sense text |
| **Alignment** | Alineació: Esquerra o Dreta |

## Quan usar-lo

- Per explicacions o descripcions
- Per seccions de text simples
- Quan no necessites botó ni imatge

## Exemple

```
Title: "Sobre nosaltres"
Text: "Som un estudi creatiu especialitzat en disseny web..."
Alignment: Left
```

## Consells

:::tip TitleSolo
Activa **TitleSolo** si vols mostrar només un títol gran sense text a sota.
:::
