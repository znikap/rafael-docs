# Bloc Process

El bloc **Process** mostra targetes amb passos o fases del teu treball.

:::info Disponibilitat
Aquest bloc només està disponible a la **Front Page**.
:::

## Camps

| Camp | Descripció |
|------|------------|
| **Pretitle** | Text petit sobre les targetes (ex: "COM TREBALLO") |
| **Cards** | Llista de targetes amb els passos |

### Cada targeta (Card)

| Camp | Descripció |
|------|------------|
| **Title** | Nom del pas (ex: "Descobriment") |
| **Text** | Descripció del pas |
| **Image** | Icona o il·lustració |

## Exemple

```
Pretitle: "EL NOSTRE PROCÉS"

Card 1:
  Title: "Descobriment"
  Text: "Comencem entenent les teves necessitats i objectius"
  Image: [icona de lupa]

Card 2:
  Title: "Disseny"
  Text: "Creem propostes visuals i prototips"
  Image: [icona de llapis]

Card 3:
  Title: "Desenvolupament"
  Text: "Construïm el producte final"
  Image: [icona de codi]

Card 4:
  Title: "Llançament"
  Text: "Posem el projecte en marxa"
  Image: [icona de coet]
```

## Consells

:::tip Nombre de targetes
Recomano entre 3 i 5 targetes per mantenir la claredat.
:::
