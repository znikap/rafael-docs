# Bloc Text amb CTA

El bloc **Text amb CTA** (Call To Action) afegeix un botó d'acció al text.

## Camps

| Camp | Descripció |
|------|------------|
| **Pretitle** | Text petit sobre el títol (opcional) |
| **Title** | Títol principal |
| **Title Size** | Mida: Medium o Large |
| **Text** | Contingut del paràgraf |
| **Link** | Botó amb enllaç |

### Camp Link (botó)

| Camp | Descripció |
|------|------------|
| **Title** | Text del botó (ex: "Contacta'ns") |
| **Link** | URL de destí |

## Tipus d'enllaços

Pots usar diferents tipus d'URL:

| Tipus | Exemple |
|-------|---------|
| URL externa | `https://google.com` |
| Pàgina interna | `/contacte` |
| Àncora | `#serveis` |

## Exemple

```
Pretitle: "SERVEIS"
Title: "Disseny web a mida"
Title Size: Large
Text: "Creem llocs web únics per al teu negoci..."
Link Title: "Veure serveis"
Link URL: "/serveis"
```

## Quan usar-lo

- Per cridar a l'acció (comprar, contactar, etc.)
- Per destacar una secció amb botó
- Per enllaçar a altres pàgines
