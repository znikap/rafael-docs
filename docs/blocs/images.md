# Bloc Images (Galeria)

El bloc **Images** permet mostrar d'1 a 3 imatges en una graella.

## Camps

| Camp | Descripció |
|------|------------|
| **Images** | Puja d'1 a 3 imatges |
| **Fullscreen** | Activa vista a pantalla completa (per 1 imatge) |

## Com es mostren

| Nombre d'imatges | Disposició |
|------------------|------------|
| 1 imatge | Imatge gran centrada |
| 2 imatges | Dues imatges costat a costat |
| 3 imatges | Tres imatges en fila |

## Opció Fullscreen

Quan **Fullscreen** està activat:
- L'usuari pot clicar la imatge
- S'obre a pantalla completa
- Ideal per mostrar detalls

:::caution Nota
L'opció Fullscreen funciona millor amb una sola imatge.
:::

## Pujar imatges

1. Clica el camp **Images**
2. Tria **Upload files** per pujar noves imatges
3. O selecciona imatges ja pujades a la **Media Library**

## Mides recomanades

| Tipus | Amplada recomanada |
|-------|-------------------|
| Imatge sola | 1920px |
| 2 imatges | 960px cadascuna |
| 3 imatges | 640px cadascuna |

## Formats acceptats

- **JPG** - Per fotografies
- **PNG** - Per imatges amb transparència
- **WebP** - Format modern (millor compressió)
